¿Qué es GitHub?

GitHub es una plataforma de desarrollo colaborativo basada en la nube que utiliza el sistema de control de versiones Git. Es el lugar donde millones de desarrolladores almacenan y comparten sus proyectos, colaboran en el código, y controlan versiones de sus archivos. GitHub facilita el trabajo en equipo, permitiendo la colaboración en proyectos desde cualquier parte del mundo.

Características Claves

  • Repositorios: Los proyectos se almacenan en "repositorios", donde se guarda el código fuente y su historial de cambios.
  • Colaboración: GitHub permite la colaboración en código mediante solicitudes de extracción (pull requests), revisiones de código, y gestión de problemas (issues).
  • Integraciones: Se integra con herramientas de automatización, CI/CD (Integración continua/Entrega continua), y servicios en la nube.
  • Open Source: GitHub es ampliamente usado para proyectos de código abierto.

Fundadores de GitHub

GitHub fue fundado en 2008 por:

  • Tom Preston-Werner: Desarrollador de software, conocido por su participación en el desarrollo de Jekyll y otros proyectos open source.
  • Chris Wanstrath: Co-fundador y ex-CEO de GitHub. Actualmente es parte del equipo de tecnología en Microsoft tras la adquisición de GitHub en 2018.
  • PJ Hyett: Ingeniero de software y empresario, también co-fundador de GitHub.
  • Scott Chacon: Desarrollador de software y autor del libro "Pro Git". Fue uno de los co-fundadores clave de GitHub y contribuyó a la popularización de Git.
